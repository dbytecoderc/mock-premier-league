### Installation

Follow these steps to set up the app.

#### Clone the repo:

`$ git clone https://github.com/dbytecoderc/mock-premier-league.git`

#### Navigate to the project directory:

`$ cd mock-premier-league`

#### Install dependencies

`npm install`

#### Start the application

`npm run start:dev`
